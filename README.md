# Weather-Forecast
=======
This is a simple weather app and provides the following information:
Hourly weather for a day(upto 1 week)
Rainfall in mm for each day(upto 1 week)
Snowfall in cm for each day(upto 1 week)
wind speed in km/h for each day(upto 1 week)
Sunrise and sunset timings
Air Quality Index(AQI) Indian Standard
Population the region

Simple api calls are made inside useEffect hooks to optimize the working 
Enter the location and get all the details on the screen in no time!
